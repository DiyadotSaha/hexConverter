Diya Saha 

-----------
DESCRIPTION

This program accepts hex numbers as arguments and prints them out on the screen. 
It converts the hex numbers into interger values and prints them onto the screen. 
It also sorts the integer values into the ascending order and prints it out onto
the screen

-----------
FILES
-
Convert.asm
This file contains all the MIPS code used for the program described above.
- 
Diagram.pdf
This file contains the block diagram of the MIPS code for the program described 
above. 

-------------
INSTRUCTIONS

This program is intended to be run from the MARS IDE.
